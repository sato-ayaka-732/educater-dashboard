# プロジェクト概要

未経験エンジニア研修生の進捗管理システムを構築する。

現在はGoogleスプレッドシートで管理しているが、
研修生増加により管理コストが高くなっているため、
Webシステム化して自動化する。

対象ユーザー：
- 講師（佐藤彩夏）
- メンタリング担当
- 営業担当
- 研修生（将来的に）

技術スタック：
- Next.js（App Router / TypeScript）
- Supabase
- Vercel

# 現在管理している項目

研修生情報：

- 名前
- 所属会社
- 担当営業
- 入社月
- 担当講師
- メンタリング担当
- GitHubアカウント
- Canvaメールアドレス
- 備考
- 除外フラグ

学習進捗：

- 着手中コース
- 課題進捗
- 課題ごとの期限
- 営業着地コース予想
- 日報提出状況
- 感想

# MVP機能

1.研修生一覧
2.研修生詳細
3.課題期限管理
4.ダッシュボード

実装前に以下を必ず実施：

1.DB設計
2.画面設計
3.ディレクトリ構成提案
4.実装
